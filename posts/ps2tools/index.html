PS2 Tools

Requirements:

	- Memory Card with FreeMcBoot installed.
	- Usb drive.

Tools

PS2 Retroarch Emulators for Sega, Gameboy, Famicom, etc. Just copy and paste to usb drive. http://buildbot.libretro.com/nightly/playstation/ps2/

OPL Manager Manage roms (PS1 and PS2), add art covers. Then copy and paste to usb drive. https://oplmanager.com/site/

PSXVCD This application will help with converting PS1 games images (.zip, .7z, .rar, .ecm, .bin, .cue) into VCD to use it with POPStarter). https://www.psx-place.com/threads/psxvcd-by-englishl1989.19063/

POPStarter POPStarter is a launcher which lets you play your PS1 games in combination with Sony’s PS1 emulator for PS2 (known as “POPS” or “SLBB-00001”). https://www.psx-place.com/threads/popstarter.19139/

USBUtil Useful for moving large PS2 game files to usb FAT32 drives. https://www.psx-place.com/threads/usbutil-by-iseko.19048/

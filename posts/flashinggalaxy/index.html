Flashing Samsung Galaxy S7 Edge SM-G935FD

Requirements:

  • Samsung USB Drivers
  • Odin3 V3.13
  • TWRP
  • Files for SD Card
    • Magisk
    • No-verity-opt-encrypt
    • RMM-State-Bypass
    • Samsung Anti Root Removal

Procedure

  • Install drivers on computer.
  • On phone:
    • Enable Developer Options
    • Enable USB Debugging
    • OEM Unlock
  • Open Odin
    • Untick Auto Reboot
    • AP –> Select TWRP
    • Start phone in Download Mode (Vol Down + Menu + Power Button)
    • Start
  • Restart phone in Recovery Mode (Vol up + Menu + Power Button)
    • Wipe –> Wipe Data
  • Copy Files from SD Card Folder to Phone
  • From TWRP Install in this order:
    • No-verity-opt-encrypt
    • RMM-State-Bypass
    • Magisk
    • Samsung Anti Root Removal
  • Reboot –> System –> Don’t install
  • Update Magisk from official site.

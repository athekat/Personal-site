Siempre Pobres Clan Website.

  • Active

Visit the GitHub repository. This is a simple webpage made with Bootstrap that shows some stats from an Age Of Empires II Clan I have with some friends. The information is gathered using a Python script that interacts with external APIs (like LibreMatch and Aoe2 Companion), saves them to a json file and then is retrieved using a js script. The 1v1 elo chart is generated with Plotly using Age of Empires II: Definitive Edition Rating Charts data. Everything is automatically updated using GitHub Actions.
